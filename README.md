# dynamic-form

## Environment Variables

When configuring the backend service (`apps/microfrontends/backend`), you can use the following environment variables:

- `SCHEMA_API_URL`: The URL to fetch entity schemas.
- `SCHEMA_API_HEADERS`: A JSON string of headers to send with schema requests.
- `ENUM_API_URL`: The URL to fetch enum values.
- `ENUM_API_HEADERS`: A JSON string of headers to send with enum requests.
